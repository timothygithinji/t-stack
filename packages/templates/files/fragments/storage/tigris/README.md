# Tigris object storage

Tigris is a hosted, S3-compatible object storage service; no local emulator is
required. Configure your bucket and credentials via Doppler or env vars:

  TIGRIS_ACCESS_KEY_ID
  TIGRIS_SECRET_ACCESS_KEY
  TIGRIS_ENDPOINT_URL  (e.g. https://fly.storage.tigris.dev)
  TIGRIS_BUCKET

See https://www.tigrisdata.com/docs/ for usage details.
